ArcGIS Pro Locators Using Free Geocoding Services

Follow these steps to add a free geocoding service instead of using default Esri World Geocoder which costs 40 credits per 1000 addresses. //20170426

  1. In ArcGIS Pro, go to INSERT/Connections to add the URL to a unfederated ArcGIS Server like http://services.gis.ca.gov/arcgis/services from State GIO.
  2. Under the Project Pane/Servers folder, find the GIS Server just added and expand to the geocoding services.
  3. Right click on on the service you want to to use and click Add To Project. The locator will appear under the Locators folder, i.e. Location/NorthAmerica, and this option will now appear in the Geocode Addresses tool Input Address Locator parameter picklist. //20170426
  4. The goecoder may require fields named differently than your dataset. You can map the fields in your input table with the Alias Name field picklists, i.e. Street:Address1, ZIP:PostalCode.
